## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Amjad0202/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open browser**
   Navigate to `http://localhost:3000`
